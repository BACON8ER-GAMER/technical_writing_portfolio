AI Assisted Coding Guide for Secondary Students

This project was a collaborative effort with my classmates. Our goal was to help educate secondary students on how to properly use AI tools to assist them in their coding projects. The project was created using GitHub, Markdown, and HTML. The project is hosted directly on GitHub pages.

Git Workflow Tutorial for Technical Writers

This project was a collaborative effort with my classmates. Our goal was to help explain the Git and GitHub workflow in an easy to understand way for technical writers who are unfamiliar with Git and GitHub. For this project, I converted the original file from AsciiDoc format to HTML and CSS to created a website that is easy and intuitive to use. This project is hosted directly on GitHub pages as well.

Click here to view the project.

Tool Chooser for Technical Writers

This project was a collaborative effort and was the third project in the second course of the program. Our goal was to create a website that could help technical writers choose the right tool for their specific needs for their projects. With this project, I was the editor for the written content and tested the website for usability and accessibility.
